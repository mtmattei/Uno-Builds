# Radial Action Menu

A reusable, animated radial action menu control for Uno Platform applications.

## What is it?

This project demonstrates a custom floating action button (FAB) that expands into a radial menu with smooth, elastic animations. When tapped, menu items spring outward in an arc pattern with staggered timing and bounce effects.

## Features

- Position-aware placement (supports all four corners)
- Smooth elastic animations with GPU acceleration
- Customizable items with icons and commands
- Built with Uno Platform for cross-platform compatibility
- Material Design theming

## Demo

The demo app includes two radial menus in the bottom corners, showcasing different item counts and animations.

## Technologies

- Uno Platform
- C# / XAML
- Material Design theme
- Storyboard animations with CompositeTransform
